# Документация плагина av-image для TinyMCE

Полный анализ структуры и методов плагина av-image.js (3637 строк кода)

## Общая структура плагина

Плагин состоит из 9 основных классов и объектов:

1. **AvAssets** - объект для управления SVG иконками
2. **AvLoader** - класс для загрузки файлов и URL изображений
3. **AvPresetManager** - класс для управления пресетами изображений
4. **AvExportModal** - класс для модального окна экспорта
5. **AvImageTools** - класс для обработки изображений
6. **AvWatermarkManager** - класс для управления водяными знаками
7. **AvCropManager** - класс для обрезки прямоугольных областей
8. **AvCircleManager** - класс для обрезки круглых областей
9. **AvFrameManager** - класс для применения рамок к изображениям
10. **AvUIManager** - класс для генерации UI интерфейса
11. **AvModal** - основной класс модального окна редактора

---

## 1. AvAssets (строки 10-27)

**Назначение**: Объект для генерации SVG иконок с блокировкой

### Методы:
- `getLockIcon(isLocked)` - возвращает SVG иконку замка (зеленая/красная)

---

## 2. AvLoader (строки 30-84)

**Назначение**: Загрузка и обработка файлов изображений

### Методы:
- `loadFile(file)` - загружает файл через FileReader, возвращает Promise с dataUrl и метаданными
- `loadUrl(url)` - загружает изображение по URL с CORS, возвращает Promise

---

## 3. AvPresetManager (строки 87-126)

**Назначение**: Управление пресетами размеров изображений

### Конструктор:
- `AvPresetManager(baseUrl)` - принимает базовый URL для загрузки конфигов

### Методы:
- `load()` - загружает основные пресеты из `/configs/image-presets.json`
- `loadContainerPresets()` - загружает пресеты контейнеров из `/configs/image-presets-size-container.json`

---

## 4. AvExportModal (строки 129-265)

**Назначение**: Модальное окно для экспорта изображений с настройками

### Конструктор:
- `AvExportModal(core)` - принимает объект core с состоянием

### Методы:
- `render()` - создает HTML структуру модального окна экспорта
- `refresh()` - обновляет отображение модального окна
- `bindEvents()` - привязывает обработчики событий к элементам
- `handleClick(target, event)` - обрабатывает клики по кнопкам экспорта

---

## 5. AvImageTools (строки 268-496)

**Назначение**: Инструменты для обработки изображений (canvas операции)

### Методы:
- `resize(dataUrl, targetW, targetH)` - изменяет размер изображения
- `rotate(dataUrl, angle)` - поворачивает изображение на заданный угол
- `flip(dataUrl, direction)` - отражает изображение (horizontal/vertical)
- `getRenderedImageRect(imgElement)` - получает реальные размеры отрендеренного изображения
- `applyFrame(dataUrl, cfg)` - применяет рамку к изображению
- `drawRoundedRect(ctx, x, y, width, height, radius)` - рисует скругленный прямоугольник
- `applyShadow(dataUrl, cfg)` - применяет тень к изображению

---

## 6. AvWatermarkManager (строки 499-597)

**Назначение**: Управление водяными знаками на изображении

### Конструктор:
- `AvWatermarkManager(overlayCanvas, imgElement)` - принимает canvas для overlay и элемент изображения

### Методы:
- `draw(config)` - рисует водяной знак согласно конфигурации
- `clear()` - очищает водяной знак с overlay canvas

---

## 7. AvCropManager (строки 600-975)

**Назначение**: Управление обрезкой прямоугольных областей изображения

### Конструктор:
- `AvCropManager(overlayCanvas, imgElement)` - принимает canvas и изображение

### Методы:
- `init()` - инициализирует менеджер обрезки с базовыми размерами
- `draw()` - рисует область обрезки с ручками управления
- `drawHandle(ctx, x, y)` - рисует ручку управления в заданной точке
- `setShowGrid(show)` - включает/выключает отображение сетки
- `toggleProportional()` - переключает пропорциональный режим
- `isProportional()` - возвращает состояние пропорционального режима
- `getCropDimensions()` - возвращает размеры области обрезки
- `setTargetSize(targetWidth, targetHeight)` - устанавливает целевой размер
- `initInteraction(onUpdate)` - инициализирует интерактивность с callback
- `getHandleAtPoint(x, y)` - определяет ручку управления в точке
- `isPointInside(x, y)` - проверяет, находится ли точка внутри области
- `onMouseDown(x, y)` - обработчик нажатия мыши
- `onMouseMove(x, y)` - обработчик перемещения мыши
- `onMouseUp()` - обработчик отпускания мыши
- `getCroppedImage()` - возвращает обрезанное изображение как dataUrl
- `destroy()` - освобождает ресурсы

---

## 8. AvCircleManager (строки 977-1221)

**Назначение**: Управление обрезкой круглых областей изображения

### Конструктор:
- `AvCircleManager(overlayCanvas, imgElement)` - принимает canvas и изображение

### Методы:
- `init()` - инициализирует менеджер с круглой областью по центру
- `draw()` - рисует круглую область обрезки с ручками
- `drawHandle(ctx, x, y)` - рисует ручку управления
- `onMouseDown(x, y)` - обработчик нажатия мыши
- `onMouseMove(x, y)` - обработчик перемещения мыши
- `onMouseUp()` - обработчик отпускания мыши
- `getCroppedImage()` - возвращает круглое изображение с прозрачным фоном
- `destroy()` - освобождает ресурсы

---

## 9. AvFrameManager (строки 1149-1221)

**Назначение**: Применение декоративных рамок к изображениям

### Конструктор:
- `AvFrameManager(overlay, img)` - принимает overlay canvas и элемент изображения

### Методы:
- `draw(cfg)` - рисует рамку согласно конфигурации
- `drawRoundedRect(ctx, x, y, width, height, radius)` - рисует скругленный прямоугольник

---

## 10. AvUIManager (строки 1224-2126)

**Назначение**: Генерация пользовательского интерфейса модального окна

### Методы:
- `renderLayout(state, presetsHtml)` - создает основную структуру интерфейса
- `renderContent(state, presetsHtml)` - генерирует содержимое всех табов
- `renderPresetCard(preset, index)` - создает карточку пресета
- `injectStyles()` - вставляет CSS стили в документ

### Особенности:
- Содержит огромный блок CSS стилей (800+ строк)
- Генерирует сложный HTML для всех элементов интерфейса
- Поддерживает табы: Upload, Size, Crop, Circle, Watermark, Frame, Shadow

---

## 11. AvModal (строки 2129-3637)

**Назначение**: Основной класс модального окна редактора изображений

### Конструктор:
- `AvModal(editor)` - принимает экземпляр TinyMCE редактора

### Основные свойства состояния:
- `loadedImage` - загруженное изображение (dataUrl)
- `fileInfo` - информация о файле
- `cropManager` - менеджер обрезки
- `circleManager` - менеджер круглой обрезки
- `watermarkManager` - менеджер водяных знаков
- `frameManager` - менеджер рамок

### Методы управления:
- `addLog(msg)` - добавляет сообщение в лог
- `loadLibrary()` - загружает внешние библиотеки
- `updateExportSize()` - обновляет размеры для экспорта
- `loadPresets()` - загружает пресеты размеров
- `updateIconsUI()` - обновляет иконки в интерфейсе

### Методы отрисовки:
- `create()` - создает модальное окно
- `render()` - отрисовывает содержимое модального окна
- `updatePosition()` - обновляет позиционирование элементов

### Методы overlay управления:
- `updateCropOverlay()` - обновляет overlay для обрезки
- `updateWatermarkOverlay()` - обновляет overlay для водяных знаков
- `updateCircleOverlay()` - обновляет overlay для круглой обрезки
- `updateFrameOverlay()` - обновляет overlay для рамок

### Обработка событий:
- `initEvents()` - инициализирует все обработчики событий (1000+ строк кода)
  - Обработка загрузки файлов (drag&drop, file input, URL)
  - Переключение между табами
  - Управление размерами и пресетами
  - Обработка обрезки и других инструментов
  - Экспорт в различных форматах

### Методы загрузки:
- `uploadImage(dataUrl)` - загружает изображение на сервер
- `processFile(file, source)` - обрабатывает загруженный файл

---

## Регистрация плагина

Плагин регистрирует кнопку 'av-image-text' в TinyMCE с текстом "Вставить изображение", которая создает новый экземпляр AvModal при клике.

---

## Технические особенности

1. **Размер**: 3637 строк кода
2. **Canvas операции**: Широкое использование HTML5 Canvas для обработки изображений
3. **Асинхронность**: Использование Promise для загрузки файлов
4. **Модульность**: Четкое разделение функционала по классам
5. **Интерактивность**: Поддержка drag&drop, мышиных событий, resize
6. **Форматы**: Поддержка JPG, PNG, WebP экспорта
7. **Пресеты**: Система настраиваемых размеров и конфигураций

Плагин представляет собой полноценный редактор изображений, интегрированный в TinyMCE.
