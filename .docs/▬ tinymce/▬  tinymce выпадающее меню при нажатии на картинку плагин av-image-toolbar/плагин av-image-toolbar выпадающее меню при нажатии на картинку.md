# Плагин av-image-toolbar (Выпадающее меню при нажатии на картинку)

### Обзор

Плагин `av-image-toolbar` является центральным хабом («оркестратором») для работы с изображениями в редакторе Aurora. Он регистрирует контекстную панель инструментов (Context Toolbar), которая появляется автоматически при клике на изображение (точнее, на контейнер `<figure class="aurora-image">`).

Вместо того чтобы содержать всю логику редактирования в себе, этот плагин делегирует выполнение команд специализированным микро-плагинам, таким как `av-image-seo` и `av-image-link`.

---

### Основные возможности

Тулбар предоставляет следующие быстрые действия (кнопки):

1.  **Редактировать изображение (`av-context-edit`)**: Вызывает полноценный редактор изображений (плагин `av-image`).
2.  **SEO и атрибуты (`av-context-seo`)**: Открывает легкое модальное окно для быстрой правки Alt-текста, Title и Подписи (плагин `av-image-seo`).
3.  **Ссылка и поведение (`av-context-link`)**: Открывает модальное окно для настройки ссылок (плагин `av-image-link`).
4.  **Отступы изображения (`av-image-wrapping`)**: Вызывает выпадающее меню для управления внешними отступами (из плагина `av-image-wrapping`).

---

### Архитектура и взаимодействие

Плагин спроектирован по модульному принципу. Он не обращается к другим плагинам напрямую через JS-объекты, а использует стандартный механизм команд TinyMCE — `editor.execCommand`.

**Как это работает:**

- При клике на кнопку SEO тулбар выполняет: `editor.execCommand('avImageSeo', false, selectedNode)`.
- TinyMCE ищет плагин, который зарегистрировал команду `avImageSeo`, и передает управление ему.

Это позволяет каждой части системы (SEO, Ссылки, Графика) развиваться независимо и не перегружать основной код.

---

### Подключение и настройка

Плагин должен быть зарегистрирован в конфигурации TinyMCE. В нашей системе это делается через Angular-компонент `AvTinymceControlComponent`.

#### 1. Регистрация файла

В секции `external_plugins` укажите путь к файлу:

```javascript
external_plugins: {
  'av-image': '/assets/tinymce/plugins/av-image/plugin.js',
  'av-image-toolbar': '/assets/tinymce/plugins/av-image-toolbar/plugin.js',
  'av-image-seo': '/assets/tinymce/plugins/av-image-seo/plugin.js',
  'av-image-link': '/assets/tinymce/plugins/av-image-link/plugin.js',
  'av-image-wrapping': '/assets/tinymce/plugins/av-image-wrapping/plugin.js'
}
```

#### 2. Активация плагина

Добавьте название плагина в список `plugins`:

```javascript
plugins: "... av-image-toolbar av-image-seo av-image-link ...";
```

#### 3. Отключение стандартных панелей

Чтобы избежать дублирования кнопок (например, стандартных кнопок выравнивания TinyMCE), необходимо отключить встроенные контекстные панели для изображений. Добавьте следующие параметры в объект инициализации `tinymce.init`:

```javascript
image_context_toolbar: false,
quickbars_image_toolbar: false,
```

Это гарантирует, что при клике на изображение будет отображаться **только наш** универсальный тулбар.

---

### Требования к HTML

Чтобы тулбар появился, изображение в редакторе должно иметь следующую структуру:

```html
<figure class="aurora-image" data-align="center">
  <img src="..." alt="..." title="..." />
  <figcaption>Описание...</figcaption>
</figure>
```

Плагин использует предикат (условие), который ищет тег `figure` с классом `aurora-image`.

---

### Пример кода кнопки (изнутри плагина)

```javascript
editor.ui.registry.addButton("av-context-seo", {
  icon: "av-icon-seo",
  tooltip: "SEO и атрибуты (Alt, Title)",
  onAction: function () {
    // Вызываем команду другого плагина
    editor.execCommand("avImageSeo", false, editor.selection.getNode());
  },
});
```

---

### Преимущества такого подхода

- **Производительность**: Микро-плагины для SEO и Ссылок весят копейки и открываются мгновенно, не инициализируя полноценный графический редактор.
- **Стабильность**: Мы правим только конкретные атрибуты DOM (`data-alt`, `href`), не рискуя сломать структуру всего блока `figure`.
- **UX**: Пользователь получает именно тот инструмент, который ему нужен в данный момент, без лишнего визуального шума.
