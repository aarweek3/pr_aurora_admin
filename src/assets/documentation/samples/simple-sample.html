Simplified Manager ⚡ — Базовый шаблон управления

Компонент Simplified Manager является эталонным примером реализации простого CRUD-интерфейса (Create, Read, Update, Delete) в проекте Aurora Admin. Он сочетает в себе высокую производительность, чистоту кода и использование современных возможностей Angular.

Основные возможности:
  • Поиск в реальном времени: фильтрация списка по названию.
  • Пагинация: управление размером страницы и переключение между ними.
  • Модальное управление: добавление, редактирование и просмотр данных в одном окне.
  • State Management: использование RxJS и сигналов через сервис.

1. Структура и файлы

Модуль организован по стандарту «одна папка — одна функциональность»:

  • SimpleManagerComponent: Основной контейнер. Содержит поиск, пагинацию и модальное окно.
  • SimpleTableComponent: Презентационный компонент для отображения таблицы.
  • SimpleModalComponent: Презентационный компонент для формы редактирования.
  • SampleSimpleService: Сервис состояния (бизнес-логика).
  • SampleSimpleApiService: Сервис для работы с HTTP.

2. Логика работы

Вся логика вынесена в SampleSimpleService. Компонент лишь подписывается на состояние и вызывает методы сервиса. Это делает код компонента максимально компактным (около 100 строк). 

// Пример того, как компонент получает данные
public service = inject(SampleSimpleService);
state$ = this.service.getState();
