Общий обзор системы

Текущая архитектура безопасности Aurora v3.5.

Принципы

  • HttpOnly Cookies для хранения токенов.
  • JWT для аутентификации запросов.
  • Identity для управления пользователями.

Статус

Система стабильна, используется во всех модулях админ-панели. Готова к разделению на независимые сервисы.

Схема взаимодействия

Взаимодействие между Angular и ASP.NET Core происходит через защищенный канал с автоматическим продлением сессии.
