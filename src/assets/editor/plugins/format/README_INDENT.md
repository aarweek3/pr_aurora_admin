# Плагины отступов (Indent/Outdent)

## Описание

Два плагина для управления отступами блочных элементов в редакторе:
- **IndentPlugin** - увеличение отступа (→)
- **OutdentPlugin** - уменьшение отступа (←)

## Возможности

### IndentPlugin (Увеличить отступ)
- **Иконка**: → (стрелка вправо)
- **Горячая клавиша**: `Tab`
- **Действие**: Увеличивает `margin-left` текущего блока на 40px
- **Применяется к**: параграфы, заголовки, цитаты, списки

### OutdentPlugin (Уменьшить отступ)
- **Иконка**: ← (стрелка влево)
- **Горячая клавиша**: `Shift+Tab`
- **Действие**: Уменьшает `margin-left` текущего блока на 40px
- **Минимум**: 0px (отступ не может быть отрицательным)

## Поддерживаемые элементы

Плагины работают с блочными элементами:
- `<p>` - параграфы
- `<div>` - блоки
- `<h1>` - `<h6>` - заголовки
- `<blockquote>` - цитаты
- `<pre>` - предформатированный текст
- `<ul>`, `<ol>`, `<li>` - списки

## Пример использования

```typescript
// Плагины автоматически регистрируются в AuroraEditorComponent
this.plugins = [
  // ...
  new IndentPlugin(),   // Увеличить отступ
  new OutdentPlugin(),  // Уменьшить отступ
  // ...
];
```

## Как это работает

1. Пользователь ставит курсор в блок или выделяет текст
2. Нажимает кнопку или горячую клавишу
3. Плагин находит родительский блочный элемент
4. Изменяет его `margin-left` на ±40px

### Пример DOM изменений

**До:**
```html
<p>Текст без отступа</p>
```

**После IndentPlugin:**
```html
<p style="margin-left: 40px">Текст с отступом</p>
```

**После повторного IndentPlugin:**
```html
<p style="margin-left: 80px">Текст с двойным отступом</p>
```

**После OutdentPlugin:**
```html
<p style="margin-left: 40px">Текст с одинарным отступом</p>
```

## Интеграция в toolbar

Плагины автоматически отображаются в отдельной группе между списками и выравниванием:

```
[Bold] [Italic] | [List] [List] | [→] [←] | [Left] [Center] [Right]
                                   ↑   ↑
                            Indent  Outdent
```

## Особенности

- ✅ Работает с вложенными элементами
- ✅ Поддерживает множественные уровни отступа
- ✅ Безопасно (минимум 0px для outdent)

- ✅ Сохраняет форматирование при undo/redo
