[ ⚙️ Реактивность (Signals) ]
------------------------------------------
Все модули v3.5 обязаны использовать Angular Signals. Состояние хранится в signal<State> внутри StateService. Прямые манипуляции с переменными в компонентах запрещены — только через методы State.

[ 🌍 Локализация (English Fallback) ]
------------------------------------------
Механизм English Fallback (Эталон)Система Aurora v3.5 минимизирует рутинную работу по заполнению переводов. Если пользователь не заполнил вкладку другого языка, данные будут автоматически подтянуты из эталонной английской версии.

Приоритеты Fallback:Значение в текущей вкладке.
Значение из вкладки en-US.
Значение из технического поля Name (Техническое название).
Эталонная реализация (StateService):applyEnglishFallbacks(localizations: any[], technicalName: string): any[] {
  const enLocale = localizations.find(l => l.languageCode === 'en-US');
  return localizations.map(loc => ({
    ...loc,
    name: loc.name || enLocale?.name || technicalName,
    description: loc.description || enLocale?.description,
    metaTitle: loc.metaTitle || enLocale?.metaTitle,
    metaDescription: loc.metaDescription || enLocale?.metaDescription
  }));
}

// ...

openHelp() {
  this.http.get('assets/help-data/your-file-id.json').subscribe((data: any) => {
    this.modal.create({
      nzTitle: undefined,
      nzContent: HelpUniversalModalComponent,
      nzData: { ...data, initialMode: 'view' },

[ 🖼️ Медиа: Алфавитное шардирование ]
------------------------------------------
Путь: uploads/{folder}/{shard}/{filename}

C# GetShard Logic:private string GetShard(string filename) {
  char first = char.ToLower(filename[0]);
  return (first >= 'a' && first <= 'z') ? first.ToString() : "0";
}

[ 🛠️ Обслуживание (Maintenance) ]
------------------------------------------
Seed: Загрузка данных из JSON. Блокируется 409 Conflict, если БД не пуста (count > 0).
Clear: Полная очистка таблиц. Требует прохождения капчи (modal.challenge) на фронтенде.
