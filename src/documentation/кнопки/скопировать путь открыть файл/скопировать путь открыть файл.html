Руководство: Кнопки «Скопировать путь» и «Открыть в редакторе»

Данный функционал предназначен для ускорения разработки и отладки. Он позволяет разработчику мгновенно скопировать путь к файлу или открыть его в IDE прямо из браузера.

Зачем это нужно?
  • Не нужно искать файл вручную в дереве проекта.
  • Исключаются ошибки при ручном вводе путей.
  • Мгновенный переход к редактированию документации или кода компонента.

1. Как это работает (Frontend)

Для реализации используется метод openFile(path: string) в сервисе IconLaboratoryService. Он отправляет POST-запрос на бэкенд с путем к файлу.

// Пример вызова в компоненте
this.iconLabService.openFile('src/documentation/sample.html').subscribe();

2. Как это работает (Backend)

Бэкенд принимает путь, проверяет его на безопасность (разрешены только пути внутри d:\_PROGECT\) и запускает процесс через стандартную оболочку Windows:

Process.Start(new ProcessStartInfo { 
    FileName = fullPath, 
    UseShellExecute = true 
});
⚠️ Важно: Если файл не открывается, убедитесь, что расширение файла (.ts, .html, .md) ассоциировано в Windows с вашим редактором (например, VS Code).

4. Промпты для AI-ассистента (Antigravity)

Скопируйте и используйте эти промпты, когда вам нужно будет внедрить аналогичные кнопки в новые части проекта:

Для Angular-компонента:
«Добавь блок технической информации (отладки) в этот компонент по аналогии с HelpDocumentationComponent (src/app/pages/help/help-documentation/help-documentation.component.ts). Нужно вывести путь к файлу текущего компонента и путь к [название/описание файла], добавить кнопки «Скопировать путь» и «Открыть в редакторе». Используй IconLaboratoryService.openFile(), иконки copy и folder-open, и добавь тултипы. Стилизуй кнопки (синий ховер для копирования, зеленый для открытия) и упакуй блок в класс debug-info-block.»
Для HTML-файла документации:
«Вставь в этот HTML-файл блок с кнопками открытия и копирования пути. Путь должен указывать на [указать путь, например: src/documentation/...]. Оформи это как визуальный пример в рамке, используя IconLaboratoryService для открытия. Добавь иконки и тултипы в стиле Aurora Admin.»
