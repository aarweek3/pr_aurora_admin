Toggle — это универсальный компонент переключателя, поддерживающий три режима использования:

  • Directive (avToggle): Применяется к нативному <input type="checkbox">.
  • Component (<av-toggle>): Готовое решение с поддержкой внешних лейблов и позиционирования.
  • Labeled (<av-toggle-labeled>): Стильный переключатель с подписями внутри трека.

Используйте кнопку "?" в заголовке контейнера ниже, чтобы увидеть полную спецификацию и описание всех параметров.
