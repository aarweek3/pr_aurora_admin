Implementation Guide

Пошаговое руководство по использованию компонента:

1

Import Component

Импортируйте PhoneInputComponent в ваш компонент

import { PhoneInputComponent } from '@shared/components/ui';
2

Add to Imports

Добавьте компонент в массив imports

imports: [PhoneInputComponent, ReactiveFormsModule]
3

Create FormControl

Создайте FormControl для управления значением

phoneControl = new FormControl('', Validators.required);
4

Use in Template

Используйте компонент в шаблоне с FormControl

<av-phone-input [formControl]="phoneControl" [defaultCountry]="'UA'"></av-phone-input>

📋 Output Format

Компонент возвращает полный номер телефона с кодом страны в международном формате E.164:

Ukraine: +380501234567
USA: +15551234567
Poland: +48123456789
