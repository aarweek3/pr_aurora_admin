📦 PickerComponent

Комплексный контрол для управления выбором цвета. Поддерживает работу с палитрами, системными диалогами, а также предоставляет удобный интерфейс для работы с прозрачностью и форматом HEX.

🎯 Основные возможности:
  • Три режима работы: Кастомная палитра, Системный Picker, Комбинированный
  • Поддержка именованных категорий для кастомных цветов
  • Интеллектуальное управление прозрачностью и альфа-каналом
  • Адаптивный дизайн и поддержка темной темы из коробки
