📦 IconSettingsControlComponent

Комплексный контрол для управления всеми аспектами отображения иконок. Предоставляет интуитивный интерфейс для настройки размеров, цветов, трансформаций и стилизации контейнера.

🎯 Основные возможности:
  • Выбор иконки из библиотеки (более 100 иконок)
  • Настройка размера, цвета, прозрачности
  • Поворот, масштабирование, отражения
  • Стилизация контейнера (фон, рамка, отступы)

Параметры и события компонента IconSettingsControlComponent:

Параметр Тип По умолчанию Описание
value ModelSignal<AvIconConfig> - Основной сигнал конфигурации (two-way binding)
presets IconPreset[] [] Список доступных иконок для выбора
valueChange EventEmitter<AvIconConfig> - Событие изменения любого параметра

💡 Структура AvIconConfig

type: string - путь к иконке (например: "actions/av_check_mark")
size: number - размер в пикселях
color: string - цвет иконки
rotation: number - поворот в градусах
opacity: number - прозрачность (0-1)
scale: number - масштаб
flipX, flipY: boolean - отражение по осям
borderShow: boolean - показать рамку
borderColor, borderWidth, borderRadius - стили рамки
padding: number - внутренние отступы
background: string - цвет фона контейнера
