Spinner Component

Премиальные индикаторы загрузки с поддержкой анимаций, оверлеев и умных задержек.

Базовое использование

<av-spinner></av-spinner>

Внутри кнопки (inherit)

<button av-button avType="primary">
  <av-spinner size="small" color="inherit"></av-spinner>
  Loading...
</button>

Оверлей в блоке

<div style="position: relative">
  <av-spinner [overlay]="true" label="Loading..."></av-spinner>
  <!-- Контент -->
</div>

Управление видимостью (visible)

Используйте [visible] для плавного появления/скрытия без удаления из DOM:

<av-spinner [visible]="isLoading()" [delay]="300"></av-spinner>

* Рекомендуется для overlay-режимов, чтобы анимация затухания была плавной.
