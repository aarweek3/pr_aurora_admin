1. Базовое использование

Простая группа полей с заголовком:

<av-field-group label="Персональная информация">
  <input av-input placeholder="Ваше имя" />
  <input av-input placeholder="Email адрес" />
</av-field-group>

2. Сворачиваемая группа

Группа с возможностью сворачивания содержимого:

<av-field-group
  label="Дополнительные настройки"
  [collapsible]="true"
  [(isCollapsed)]="isCollapsed">
  <input av-input placeholder="Город" />
  <input av-input placeholder="Страна" />
</av-field-group>
