- [ ] Создать структуру папок: `components`, `services`, `models`.
- [ ] Создать файл `end-points.ts`.

- [ ] Реализовать `LanguageAggregatorApiService` со всеми методами (GET, POST, PUT, DELETE, PATCH для статуса).
