🚀 Справка Logger Console

🔄 История Сигналов (Signals)

Отслеживание изменений Angular Signals в реальном времени. В коде:

this.loggerService.trackSignal(mySignal, 'ИмяСигнала');

Отображает переход от старого значения к новому. Фильтр: кнопка STATE.

🖱️ Действия пользователя

Автоматическая запись всех кликов на кнопки и ссылки. Позволяет восстановить путь пользователя к ошибке. Фильтр: кнопка ACTION.

🔌 Сеть и Окружение

HTTP: Логирование всех API-запросов, их статусов и длительности. Фильтр: кнопка HTTP.

Инфо: В верхней панели отображается разрешение экрана, тема и статус сети (Online/Offline).

📦 Сервис и Экспорт

  • Export: Скачать логи в формате JSON для анализа.
  • Clear: Полная очистка текущего буфера (лимит 1000 записей).

🗄️ Storage Editor

Просмотр и редактирование LocalStorage, SessionStorage и Cookies прямо в консоли.

Вкладка 📦 Storage. Значения можно редактировать на лету — просто измените текст и нажмите Enter или кликните вне поля.

⌨️ Terminal (CLI)

Выполнение команд внутри приложения. Вкладка 💻 Terminal.

Примеры: clear-cache, help, echo hello.

🧭 Navigation Trail

Визуализация пути переходов пользователя по страницам (роутам).

Вкладка 🧭 Navigation показывает историю переходов с параметрами и временем пребывания на странице.

🏗️ Где проверить? (Песочница)

Страница «Песочницы» для проверки всех функций доступна в боковом меню:

Раздел UI Demo → Logger Console 🛠️

Прямая ссылка: http://localhost:4200/ui-demo/logger-console
