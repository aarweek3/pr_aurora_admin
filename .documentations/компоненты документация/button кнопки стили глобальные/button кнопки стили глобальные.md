# Глобальные стили кнопок Aurora (2026 Standard)

## 1. Примеры кнопок с иконками (Custom SVG)

При использовании кастомных SVG-иконок из нашей библиотеки, помещайте SVG-код непосредственно внутрь кнопки перед текстом.

### Hero Button с иконкой
*Используется для основных, завершающих действий в формах. Иконка располагается слева от текста.*
```html
<button class="av-btn-hero">
  <!-- Вставьте SVG код иконки здесь -->
  <svg width="24" height="24" viewBox="0 0 24 24">...</svg>
  СОХРАНИТЬ
</button>
```

### Secondary Button с иконкой
*Используется для действий отмены или возврата. Кнопка имеет более спокойный дизайн, чтобы не отвлекать от основной цели.*
```html
<button class="av-btn-secondary">
  <svg width="20" height="20" viewBox="0 0 24 24">...</svg>
  Отмена
</button>
```

### Кнопка-иконка (Без текста)
*Применяется в списках или тулбарах, когда действие понятно без слов (например, "Добавить", "Удалить").*
```html
<button class="av-btn av-btn--primary av-btn--shape-circle">
  <i nz-icon nzType="plus"></i>
</button>
```

## 2. Основные классы кнопок

### 1. Hero Button (`.av-btn-hero`)
Предназначена для главных действий на странице или в модальных окнах (Сохранить, Подтвердить, Отправить).

*   **Класс**: `av-btn-hero`
*   **Стиль**: Крупная кнопка (48px), скругление 12px, выраженная тень, текст в ВЕРХНЕМ РЕГИСТРЕ.
*   **Применение**: Основное действие формы, финальное подтверждение.

```html
<button type="submit" class="av-btn-hero">СОХРАНИТЬ</button>
```

### 2. Secondary Button (`.av-btn-secondary`)
Используется для отмены действий или второстепенных операций.

*   **Класс**: `av-btn-secondary`
*   **Стиль**: Белый фон, тонкая рамка, серый текст, скругление 12px. При наведении подсвечивается основным цветом.
*   **Применение**: Кнопки "Отмена", "Назад", "Закрыть".

```html
<button type="button" class="av-btn-secondary">Отмена</button>
```

### 3. Стандартная система кнопок (`.av-btn`)
Для рядовых действий в интерфейсе (фильтры, мелкие кнопки в таблицах).

*   **Базовый класс**: `av-btn`
*   **Модификаторы**:
    *   `.av-btn--primary` — синяя (стандартная)
    *   `.av-btn--default` — белая
    *   `.av-btn--danger` — красная
    *   `.av-btn--text` — без фона и границ

```html
<!-- Обычная синяя кнопка -->
<button class="av-btn av-btn--primary">Действие</button>

<!-- Маленькая красная кнопка -->
<button class="av-btn av-btn--danger av-btn--small">Удалить</button>
```

## 3. Управление цветами

Цвета кнопок в Aurora не задаются жестко («хардкодом»). Они привязаны к **Дизайн-токенам**, что позволяет менять тему приложения (например, на темную) без переписывания CSS кнопок.

### Основные переменные цвета:
*   `var(--av-color-primary)` — основной синий цвет кнопок Hero и Primary.
*   `var(--av-color-success-deep)` — глубокий изумрудно-зеленый цвет для позитивных действий.
*   `var(--av-color-white)` — цвет текста на темном фоне и фон кнопок Secondary.
*   `var(--av-color-error)` — цвет для кнопок удаления (`.av-btn--danger`).

### Одна и та же кнопка в разных цветах (Сравнение)

Ниже приведены примеры кнопки «СОХРАНИТЬ» в трех вариантах исполнения. Выбирайте цвет в зависимости от контекста действия.

#### 1. Стандартный (Primary Blue)
*Используется по умолчанию для большинства форм.*
```html
<button class="av-btn-hero">СОХРАНИТЬ</button>
```

#### 2. Позитивный / Успешный (Deep Green)
*Используется для подтверждения важных успешных операций или финализации настроек.*
```html
<button class="av-btn-hero" style="background: var(--av-color-success-deep); box-shadow: 0 4px 12px rgba(var(--av-color-success-deep-rgb), 0.2);">
  СОХРАНИТЬ
</button>
```

#### 3. Критический (Danger Red)
*Используется только если сохранение данных может привести к удалению или перезаписи важной информации.*
```html
<button class="av-btn-hero" style="background: var(--av-color-error); box-shadow: 0 4px 12px rgba(255, 77, 79, 0.2);">
  СОХРАНИТЬ
</button>
```

## 4. Дизайн-токены (Основа стиля)

Все кнопки используют глобальные переменные из `src/styles/abstracts/_tokens.scss`. Это позволяет изменить стиль всей системы кнопок в одном месте.

| Токен | Значение | Описание |
|-------|----------|----------|
| `--av-radius-lg` | 12px | Скругление углов для Hero-кнопок |
| `--av-color-primary` | #1890ff | Основной цвет бренда |
| `--av-motion-ease` | cubic-bezier(...) | Плавность анимации при наведении |
| `--av-spacing-lg` | 24px | Горизонтальные отступы внутри кнопок |

## Принципы использования

1.  **Одна Hero-кнопка**: На одном экране/в одной модалке должна быть только **одна** кнопка `.av-btn-hero`. Это фокусирует внимание пользователя на главном действии.
2.  **Заблокированное состояние**: Используйте стандартный атрибут `[disabled]`. Глобальные стили автоматически сделают кнопку блеклой и неактивной.
3.  **Регистр текста**: 
    *   Для `.av-btn-hero` текст всегда пишется в `ВЕРХНЕМ РЕГИСТРЕ`.
    *   Для остальных кнопок — `Как в предложении`.
